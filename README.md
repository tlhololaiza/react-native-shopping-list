## Setup Instructions

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Expo CLI

### Installation

1. Create the Expo app:
```bash
npx create-expo-app@latest react-native-shopping-list
cd react-native-shopping-list
```

2. Reset the project:
```bash
npm run reset-project
# Select 'n' when prompted
```

3. Install dependencies:
```bash
npm install @reduxjs/toolkit react-redux @react-native-async-storage/async-storage
```

4. Copy all the code files into their respective folders as shown in the folder structure.

5. Start the development server:
```bash
npx expo start
```

6. Run on your device:
   - Press `i` for iOS simulator
   - Press `a` for Android emulator
   - Scan QR code with Expo Go app on your phone
